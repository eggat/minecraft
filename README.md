# ⚔️ Minecraft 附魔報價系統 (Minecraft Enchantment Pricing System)

這是一個專為 Minecraft 伺服器設計的**純前端附魔報價系統**。
旨在讓玩家能夠直覺地挑選裝備、配置附魔，並自動計算總價格與產生報價單。系統內建了原版附魔與自定義特殊附魔（基於 ExcellentEnchants）的完整互斥與防呆邏輯。

---

## ✨ 核心特色功能

* **🛡️ 智慧裝備分支建立**：
    * **盔甲類**：點擊後提供「鑽石 / 獄髓」材質二選一，選定後再展開附魔庫。
    * **工具與武器類**：自動判定並建立「獄髓」基底，直接進入附魔配置，體驗極致流暢。
* **🚫 雙重防呆互斥邏輯**：
    * **單件互斥**：完美還原 Minecraft 原版設定（如：無限與經驗修補互斥、絲綢之觸與幸運互斥）。
    * **全域特殊附魔排斥**：特殊附魔具有「全域唯一性」，只要在一件裝備上配置了某特殊附魔，其他裝備即無法重複購買，保護伺服器經濟平衡。
* **🛒 獨立卡片式購物車 (Carousel UI)**：
    * 支援同時建立無限多件裝備（例如：同時報價兩把不同的劍）。
    * 左右切換卡片檢視不同裝備的配置與單件小計。
* **💡 直覺的操作體驗**：
    * 附魔卡片支援 **Hover Overlay（懸停特效）**，滑鼠移入即可檢視附魔詳細說明。
    * 支援「⭐ 收藏附魔」與「🕒 最近使用」分類，方便玩家快速挑選常用附魔。
* **📤 多元報價匯出**：
    * 一鍵「複製報價」至剪貼簿。
    * 支援匯出排版精美的 `TXT` 純文字檔。
    * 支援匯出資料淨化過的 `JSON` 格式，方便伺服器管理員或 Discord 機器人串接讀取。
* **🌙 極致深色主題與響應式設計**：
    * 充滿電競與科技感的 Dark Theme，帶有毛玻璃與漸層光影特效。
    * 完美支援手機、平板與電腦跨平台操作。

---

## 🛠️ 技術架構

本專案採用**完全純前端架構**，無須後端伺服器或資料庫：
* **HTML5 / CSS3**：介面佈局與流暢的過渡動畫。
* **Vanilla JavaScript (ES6+)**：核心邏輯與模組化開發，無依賴任何外部前端框架（如 React / Vue）。
* **JSON**：作為附魔資料庫（`data/enchants.json`）。
* **LocalStorage**：將玩家的購物車、收藏狀態保存在瀏覽器本地，重新整理不遺失資料。

## 🚀 安裝與執行方式

⚠️ **重要提示 (CORS 安全性限制)**：
因為系統使用 JavaScript 的 `fetch()` API 來非同步讀取 `data/enchants.json`，**請勿直接雙擊 `index.html` 開啟**（網址列若為 `file:///` 會被瀏覽器阻擋並顯示錯誤）。

### 推薦執行步驟：
1. 下載或 Clone 本專案至電腦。
2. 使用 **VS Code** 開啟專案資料夾。
3. 安裝 VS Code 擴充功能：**Live Server**。
4. 右鍵點擊 `index.html`，選擇 **"Open with Live Server"**。
5. 瀏覽器會自動開啟 `http://127.0.0.1:5500`，即可正常載入附魔資料並開始使用！
*(註：若要部署至正式環境，只需將整包檔案放上 GitHub Pages、Vercel 或任何靜態網頁伺服器即可。)*
